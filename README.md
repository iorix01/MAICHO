# Love Letter App 💌

A romantic Next.js app — a love letter page for her (the doctor).

## Install & Run

```bash
# 1. Extract the zip and enter the folder
cd love-letter-app

# 2. Install dependencies
npm install

# 3. Run in development
npm run dev
```

Then open [http://localhost:3000](http://localhost:3000) in your browser.

## Build for production

```bash
npm run build
npm start
```

## What's inside

- Landing page with animated envelope + stars background
- "Click here" button opens the envelope with animation
- 5 heartfelt letters that type out word by word
- Navigate between letters with Prev / Next
- Dark romantic theme with Cormorant Garamond font
- Fully mobile-friendly

## Stack

- Next.js 14 (App Router)
- React 18
- TypeScript
- CSS Modules (no Tailwind needed)
